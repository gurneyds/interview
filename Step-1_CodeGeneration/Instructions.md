# In this exercise you will create some code using AI

## From the Product Manager
We get historical names from another team. We need to be able to determine if these names can be taken to the Temple or not. If they can be taken to the Temple, then we need to make them available to the Ordinance Ready team so that the names can be reserved and actually taken to the Temple. The Ordinance Ready team is willing to work with us, so let's propose an interface.

Can you build this system?

Feel free to ask questions.

## Note: The team providing the historical names has provided a mock system

The `NameProvider` folder contains some code that will be a mock of the actual system. You can use this system to help with your development. Follow the instructions to get this mock system up and running locally.

## Note: Requirements for a person to be able to be taken to the Temple

1. Must have a birth year
2. Must be at least 110 years old
3. Must have a name (either first or last)
4. Must have a specified gender (male or female)
